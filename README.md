
# Educação Sustentavel

Plataforma de ensino sustentavel para o projeto da materia modelagem e Padroes de projetos


## Deploy

Para o deploy:

```bash
  npm install
  
  npm run build

  npm run dev

  php artisan storage:link
```


## Stack utilizada

**Front-end:** JS, VITE, SCSS

**Back-end:** Node, PHP
